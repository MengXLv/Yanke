# 基于 Spring Boot 的社区论坛系统

涵盖内容创作、社交互动、全文检索、积分系统四大模块，重点解决缓存一致性、高效查询、秒杀超卖等问题。

## 技术栈

Spring Boot、MyBatis-Plus、MySQL、Redis、Elasticsearch、Kafka、Docker 等

## 核心职责

- **多级缓存设计**：实现 Caffeine + Redis + MySQL 三级缓存架构，采用 Cache-Aside 写策略，热点帖子逻辑过期预热，JMeter 压测达 900+ QPS，平均响应时间 91ms；
- **缓存三防**：TTL 20% 随机抖动防雪崩、Redisson 分布式锁 + 双重检查防击穿、空值标记防穿透；
- **积分秒杀模块**：基于 Redis Lua 脚本实现库存原子扣减，库存懒加载 + DB 异步同步方案，杜绝超卖；
- **分布式基础能力**：Redis Lua + ZSet 滑动窗口限流，雪花算法生成全局唯一 ID，`@Async` 自定义线程池事务异步化；
- **全站 N+1 查询优化**：将循环单查改造为批量查询，单接口 SQL 执行次数从 21 次降至 2 次，同步修复同类问题 6 处；
- **全文检索**：Elasticsearch 7.17 + IK 中文分词器，Bool Query 实现多字段加权搜索，BulkRequest 批量构建索引。

## 项目成果

- 实现安全登录，邮件注册激活 + BCrypt 密码加密 + Token 鉴权，支持分角色管理；
- 支持帖子与社交功能，包括发帖/评论/点赞/关注/好友、热榜排序、全文检索等；
- 实现积分商城模块，包括签到/发帖/评论赚积分、商品兑换、限时秒杀。
